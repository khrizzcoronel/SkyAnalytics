## Why
Los desarrolladores externos (clientes) necesitan un Sandbox para probar la API antes de integrarla, mejorando la Developer Experience y disminuyendo el tiempo de adopción.

## What Changes
- `frontend/app/api/v1/tactico/devex/sandbox/route.ts`: Endpoint Next.js simulando respuestas del Sandbox para Mintlify.

## Capabilities
- `devex-sandbox`: Generación de datos de prueba (mocks) interactivos para documentación.
